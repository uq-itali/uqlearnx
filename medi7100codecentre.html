Welcome to your code centre for MEDI7100

This site is dedicated to the editors of MEDI7100 LearnX sites. These sites have Bootstrap (CSS and JS library) and Font Awesome (icons) referenced, and allow for some advanced formatting and icon usage. This can seem tricky and intimidating at first, so this site gives you

  • an easy to navigate menu at the top of screen to go to the element snippet or builder you require.
  • a range of code snippets (headings, image/text combinations, accordions, quotes, buttons, alerts.
  • code builders - these are powerful web applications that allow you to input plain text, rich text and links to achieve your format rather than copying and pasting code and then editing, you build your element, then copy the code created and paste it into LearnX
  • Help - if you see green question mark icons like this -   - you can hover over them fo quick guidance on what information is required.

The format and intended use of this site is as follows:

  • Each page has an example with a copy button located underneath the example. Click the button to copy the HTML required for the element.
  • Once you paste your code be sure to replace any placeholders for text, images, videos or other elements within the correct HTML tags.
  • If you encounter issues or have a suggestion for improving this page please hit the button to jump out to the Microsoft Form and log a bug/issue/suggestion in the footer of each page.

Some important things to note:

  • All elements here use colours and formatting that align with the style and approach of the MEDI7100 design team. Deviate from them at your own peril! (Only kidding, just get in touch for support and we will likely add your idea to the bunch here if it is a sound idea).
  • All placeholder images and videos are public links. They will display their content until you delete or replace them.
  • Currently, we do not have a way of pulling existing content into this site and reformmating it. We are working on it, but it is a complex task.
  • Elements are only placed here if they are tested and work.
  • The team have only piloted this in Year 1, 2022. They are still learning about how this all interacts on LearnX, so if you use this code and it goes wrong, and you ask for support, be patient! Perfection takes time, focus and diligence.
  • The format of pages is to show an example with the copy button displayed underneath the example. Click the button to copy the code for the element you want
